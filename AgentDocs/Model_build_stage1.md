We first stabilized the CNN supervised baseline and made the training, evaluation, and BotZone deployment pipelines consistent. The CNN baseline achieved 73.04% top-1, 94.90% top-3, and 98.61% top-5 accuracy on the held-out 10% replay split, with zero illegal action rate. Error analysis showed that most high-impact mistakes occurred in response decisions such as Pass/Chi/Peng/Gang confusion. To improve representational capacity, we implemented a 6-block ResNet policy model with 2.42M parameters while preserving the same 235-action masked output interface. The next stage will add a Transformer-based history encoder to incorporate public action sequences and improve response-phase decisions.
