English version:

```text
In Stage 2, we first introduced a Rank-Aware ResNet to better model Mahjong tile topology. The encoder uses rank-only convolutions, a separate honor-tile branch, and suit context mixing to avoid the misleading spatial adjacency introduced by ordinary 3x3 convolutions. RARN_m achieved lower validation loss and slightly higher top-1 accuracy than plain ResNet, but did not show a statistically significant arena advantage over ResNet. We then added a 442-dimensional public state side-channel while keeping the RARN backbone largely unchanged. The public features include visible tile counts, estimated remaining counts, per-player discards and melds, opponent meld patterns, game stage, self shanten, and effective tile estimates. RARN_public_v1 achieved 82.39% top-1, 97.75% top-3, and 99.51% top-5 accuracy with zero illegal action rate, substantially outperforming CNN, plain ResNet, and RARN_m. Per-action analysis showed that public features significantly reduced Pass/Chi/Peng/Gang confusion, making public state modeling the most effective improvement so far.
```
