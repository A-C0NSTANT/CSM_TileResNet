# Checkpoints

Large model checkpoints are not tracked in git.

Recommended local layout:

```text
checkpoints/
??? rarn_public_v2/
?   ??? rarn_public_v2_model_latest.pkl
??? rarn_public_v2_large_aug/
    ??? rarn_public_v2_large_model_latest.pkl
```

For public sharing, upload final checkpoints to GitHub Releases or external storage and link them from the main README.
